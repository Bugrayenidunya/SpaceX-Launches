# Welcome to SpaceX-Launches!
iOS App project that includes SpaceX Launch data. Used SpaceX-API.

## Usage

You don't need any special configuration. Just clone project or download to zip file and run!

## Requirements

 - iOS 14+
 - Swift 5+
 - Xcode 12+
